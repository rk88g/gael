# Carta de amor oscura para Vercel

Primera version de una web romantica, oscura y animada para dedicarle a tu novio.

## Archivos principales

- `index.html`: estructura y contenido de la pagina.
- `styles.css`: diseno, fondos, corazones, animaciones y responsive.
- `script.js`: corazones flotantes y pequenos detalles interactivos.

## Personalizacion rapida

Si quieres dejarla totalmente a tu estilo, cambia estos textos dentro de `index.html`:

- El titulo principal en la seccion hero.
- La carta dentro de la seccion `#letter`.
- Las tarjetas de "Todo lo que amo de ti".

## Subir a Vercel

1. Sube este repo a GitHub.
2. Entra a Vercel y conecta el repositorio.
3. Vercel detectara que es un sitio estatico y lo publicara sin configuracion extra.

## Desarrollo local

Puedes abrir `index.html` directamente en el navegador o levantar cualquier servidor estatico simple.
